# Concept B — design evolution comps (2026-06-26)

Directional comps exploring an evolved look & feel for the Georgette Reynolds site.
Goal: bigger type, more generous spacing, a stronger imagery-led story, and a palette
that flatters Georgette's (warm-toned) photo instead of fighting it — while still
reading as an evolution of the current site, not a reinvention.

These are **directional comps** (hero + a couple of sections, desktop only). They are
NOT the production page — `../concept-b.html` remains the live build until a direction
is chosen and built out fully (all sections, responsive, accessible).

## Files
- `option-a-warm-ivory.html` — Warm ivory canvas, forest as accent, brass hairline; Spectral. Photo sits naturally; clean and professional.
- `option-b-editorial-bold.html` — Full-bleed photo hero, Fraunces serif, cream + cognac, dark stat strip. Boldest / most imagery-led.
- `option-c-evolved-forest.html` — Keeps green dominant; portrait gets a forest duotone to harmonize. Closest to today.
- `refined-warm-ivory.html` — **Recommended.** Refined hybrid of A: warm ivory + forest + brass, split hero with an overlapping credential card, a forest "difference" band, and a services grid.

## Next step
Build `refined-warm-ivory` out as the full evolved `concept-b` — every section, responsive,
with the accessibility foundations (contrast, focus, keyboard FAQ, alt text, WebP) baked in.
